# NLP Preprocessing and Text Classification

## Objective
The objective of this project is to implement NLP preprocessing techniques and build a text classification model using machine learning.

## Dataset
SMS Spam Collection Dataset

## Techniques Used

### NLP Preprocessing
- Tokenization
- Stopword Removal
- Stemming
- Lowercasing
- Removing special characters

### Vectorization
- TF-IDF (Term Frequency - Inverse Document Frequency)

### Model
- Logistic Regression

## Steps Performed
1. Loaded dataset
2. Cleaned and preprocessed text
3. Converted text into numerical form using TF-IDF
4. Split data into training and testing
5. Trained Logistic Regression model
6. Evaluated using accuracy and classification report

## Results
- Achieved high accuracy (~95%+)
